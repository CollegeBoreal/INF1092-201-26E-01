Lors de cet atelier, nous avons découvert les bases de Git, un système de gestion de versions utilisé pour suivre les modifications des fichiers et faciliter le travail en équipe.

Nous avons appris les principales commandes de Git, comme git init, git status, git add, git commit, git pull et git push. Nous avons également compris les notions de repository (repo), commit, branche (branch) et historique.

Enfin, nous avons vu les bonnes pratiques d'utilisation de Git, les erreurs les plus courantes et la manière de collaborer efficacement sur un même projet.
