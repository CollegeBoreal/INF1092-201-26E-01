Lors de ce laboratoire, nous avons procédé à l’installation d’un disque NVMe de 1 To sur un serveur HP DL360 G6 à l’aide d’un adaptateur PCIe. Une fois le matériel installé, nous avons démarré le système d’exploitation Windows Server 2022 Datacenter afin de vérifier la reconnaissance du disque par le système.

Après confirmation de sa détection, nous avons initialisé le disque, puis effectué son formatage. Enfin, nous avons réalisé différents tests pour s’assurer de son bon fonctionnement et de sa stabilité.
