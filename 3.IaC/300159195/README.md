# Rapport d'atelier

- ID : 300159195
- Projet : 3.IaC
- Cours : INF1092-201-E26-01
- Sujet : Vérification serveur HP DL360 G6

## Description

Dans cet atelier, nous avons vérifié un serveur HP DL360 G6 afin de savoir s’il peut être acheté et utilisé dans un petit environnement informatique. L’objectif était de confirmer que le serveur fonctionne correctement avant l’achat.

Nous avons commencé par une inspection physique du serveur. Nous avons vérifié l’état du châssis, les ventilateurs, les alimentations, les disques et les ports réseau. Ensuite, nous avons démarré le serveur pour vérifier que le boot fonctionne correctement et que le BIOS est accessible.

Nous avons aussi vérifié les composants principaux comme le CPU, la RAM, les disques et le contrôleur RAID. Le RAID est important, car il permet de protéger les données en cas de panne d’un disque. Pour ce serveur, le RAID 5 est une bonne solution si au moins trois disques sont disponibles.

Enfin, nous avons analysé si le serveur peut être utilisé pour la virtualisation. Même si le DL360 G6 est un ancien serveur, il peut encore être utile pour un laboratoire, des tests ou un petit projet de formation.

## Décision finale

Le serveur peut être acheté si le démarrage est correct, si le RAID est sain, si les disques sont fonctionnels, si la RAM ne contient pas d’erreur et si les alimentations fonctionnent correctement.

Il faut refuser le serveur si un disque est en erreur, si le RAID est défectueux, si le serveur est instable ou si une erreur matérielle importante apparaît.
