1. Objectif
Installer et configurer initialement Windows Server 2022 en utilisant le démarrage réseau PXE (Preboot Execution Environment), sans support physique (clé USB).

2. Incident technique
Mon groupe a été arrêté par un problème matériel sur notre machine. Pour ne pas perdre le fil de l'atelier, j'ai rejoint un autre groupe afin de suivre et compléter les manipulations.
  
4. Étapes clés réalisées

Sélection : Choix de la version Windows Server 2022 Standard (Expérience de bureau).

Déploiement : Partitionnement du disque, copie des fichiers à travers le réseau, et redémarrage automatique.

Post-Installation : Configuration du mot de passe Administrateur et accès au Gestionnaire de serveur.

4. Conclusion
Malgré la panne matérielle de mon groupe, le fait de suivre l'autre équipe m'a permis de comprendre le processus de déploiement réseau (PXE), une méthode essentielle pour l'automatisation en entreprise.
