
Lors de ce laboratoire, j’ai installé Microsoft Windows Server à l’aide de Windows Deployment Services (WDS). Tout d’abord, j’ai démarré l’ordinateur à partir du réseau en utilisant le démarrage PXE. Ensuite, j’ai sélectionné la langue et la disposition du clavier, puis je me suis connecté au serveur de déploiement. Après cela, j’ai choisi le disque de destination et lancé l’installation du système d’exploitation. Les fichiers ont été copiés et Windows Server a été installé automatiquement. Une fois l’installation terminée, j’ai accepté les termes de la licence et créé le compte Administrateur avec un mot de passe sécurisé. Enfin, la configuration a été finalisée et le serveur était prêt à être utilisé.
