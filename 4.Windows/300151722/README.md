Rapport d'atelier - Installation de Windows Server 2022

ID : 300151722

Projet : 4.Windows

Cours : INF1092-201-E26-01



Lors de cet atelier, nous avons effectué l'installation de Windows Server 2022 sur un serveur. Nous avons d'abord vérifié les composantes matérielles du serveur, notamment les processeurs, la mémoire RAM et les disques durs.



Par la suite, nous avons démarré le serveur et configuré le démarrage à partir du média d'installation. Nous avons suivi les différentes étapes de l'assistant d'installation de Windows Server 2022 jusqu'à la création du compte administrateur.
